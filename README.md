# Avakids by @avanet
Evento con Lilypad

## Lista de accesorios:

* 1 - Lilypad arduino USB.
* 3 - LED tri-color.
* 5 - LED naranjas.
* 5 - LED Verdes.
* 1 - Lilypad button.
* 1 - lilypad switch.
* 1 - lilypad sensor de luminicidad.
* 3 - lilypad celda de bateria.
* 2 - baterias de 3 voltios.
* 1 - tambor de hilo conductor.
* 1 - set de seis cocodrilos.
* 1 - cable micro USB.
* 1 - set se seis fomies.
* 5 - cajas de almacenamiento.
